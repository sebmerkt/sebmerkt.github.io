Hi, I'm Sebastian.

I'm a Physics PhD gone full time coder. Starting out in Experimental Particle Physics at the ATLAS experiment at CERN, large scale data processing and analytics was my daily bread and butter. As an Insight Data Engineering Fellow I transformed this passion into my primary focus and I am now building ETL pipelines on AWS and my little Raspberry Pi cluster.

I enjoy learning new tools and technologies and here is the place to find out what I'm currently up to. Check out the blog section for the latest posts.

[TUTORIAL] Orchestrating an AWS EC2 cluster with Apache Airflow as a Systemd service - May 25, 2020

In this tutorial I will go over the details on how to install and run Apache Airflow as a Systemd service on an AWS EC2 instance and orchestrate a cluster. I will also describe how to use a MySQL database as a backend for Airflow. Finally, I will show how to execute ssh commands across the cluster by going over a simple example of the Airflow SSH operator.
