Pandas is a widely used Python library designed fo data analysis. It makes exploratory data analysis easy.

ROOT

ROOT is a data analysis framework developed by CERN? It is a C++ framework and widely used in High Energy Physics.

Plotly

Plotly is a visualization library for Python. It is used to produce interactive plots and figures. Check out the blog to see some examples.
